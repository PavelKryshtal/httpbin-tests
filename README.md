# API Test Automation

This project contains automated API tests written in Python using pytest.  
The tests are executed against https://httpbin.org.

## Requirements

- Python 3.11+
- Poetry

## Setup

Install dependencies:

    poetry install

## Run Tests

Run all tests with:

    poetry run pytest

## Test Report

An HTML test report is generated automatically after test execution.

Report location:

    reports/report.html

Open this file in a browser to view test results.
